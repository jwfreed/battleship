# Security Policy

## Supported Versions
Active development branch: `master`

## Reporting a Vulnerability
- Please email security reports to `security@jwfreed.dev` (or open a private GitHub security advisory).
- Include a clear description, reproduction steps, and potential impact.
- We will acknowledge receipt within 5 business days and provide status updates until resolution.

## Responsible Disclosure
Please do not create public issues for vulnerabilities until a fix is available and users have had time to upgrade.
